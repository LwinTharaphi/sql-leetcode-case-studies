# SQL LeetCode Case Studies

This repository contains SQL solutions to selected **LeetCode SQL problems**, organized by difficulty level.

---

## Folder Structure

- Each folder contains `.sql` files named using the **LeetCode problem number and title**.
- Each file includes the SQL solution for that problem.

---

## Topics Covered

- SELECT and WHERE clauses
- JOINs
- GROUP BY and HAVING
- Aggregation functions
- Subqueries
- Ranking and counting logic

---

## Notes

- Solutions use **MySQL-compatible SQL syntax**.
- Each file represents one problem and one solution.
- Focus is on clarity and correctness.
